# 公共功能要求

1. 支持SMP，公共的临界区需要使用锁保护。
2. 体系结构相关的代码放在arch/${ARCH}下面，头文件放在arch/${ARCH}/include/下面。
3. 体系结构无关的代码放在kernel/${SUBSYSTEM}下面，头文件放在include目录下面。

# 硬件能力假定

1. CPU具备64位寻址能力。
2. CPU支持MMU。
3. 支持x86-64体系结构，未来需要支持ARMv8a架构。
4. 支持SMP。

# 内核架构

1. 采用宏内核架构。
2. 要求功能组件之间接口清晰。
